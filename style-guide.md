# Guide de Style Front-end

## Mise en Page

Les designs ont été créés aux largeurs suivantes :

- Mobile : 375px
- Bureau : 1440px

## Couleurs

### Principales

- Cyan foncé : hsl(158, 36%, 37%)
- Crème : hsl(30, 38%, 92%)

### Neutres

- Bleu très foncé : hsl(212, 21%, 14%)
- Bleu grisâtre foncé : hsl(228, 12%, 48%)
- Blanc : hsl(0, 0%, 100%)
